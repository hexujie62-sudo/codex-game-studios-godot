## 本地修改

本项目将原 CCGS 体系改造为更适合 Codex 和 Godot 的工作流底座，主要修改包括：

- 移除或替换 Claude Code 生态依赖；
- 重建 Codex-only 的 Skill、Hook、路由和文档结构；
- 加入多窗口 lane、文件化 session state 和 handoff/compact/audit 流程；
- 增加 CCGS 专用 Skill 创建与测试接入方案；
- 将底层文档和高频 Skill 调整为更适合中文使用者理解的版本；
- 面向 Godot-first 游戏开发场景整理默认技术偏好。
