# Visual Work Order Template

## Screenshot Tier

- `FINAL = 240 frames`: hero final acceptance images and compare grids only.
- `REVIEW = 90 frames`: routine review images, including closeups, comparisons, burst strips, and turnarounds.
- `DEBUG = 30 frames`: diagnostic images, including ROI diffs, overlays, red/green checks, and debug material views.

All screenshots for a single work order must be captured in one engine session with the runtime capture batch mode. Report total capture time. Use explicit per-shot tier labels in the work order.

## Required Report

- Changed / unchanged.
- Uniform or parameter Diff.
- Capture list with tier and frame count.
- Total capture time.
- Engineering self-checks.
- Visual concerns awaiting external art-direction judgment.
